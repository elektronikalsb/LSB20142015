 Atencion:

 En este documento utilizaremos una plataforma creada por Google que se usa para crear aplicaciones de software para 
 sistemas operativos Android llamada App Inventor. Hemos seleccionado este sistema por que es gratuito, fácil de descargar
 y de usar y aun que las aplicaciones de App Inventor están limitadas por su simplicidad, permiten cubrir un gran numero 
 de necesidades básicas en un dispositivo móvil. La función principal de este programa consiste en leer los datos recibidos
 por el bluetooth y mostrarlos en la pantalla del dispositivo movil. Tambien, le hemos añadido un pequeño sistema de 
 seguridad para que solo el usuario pueda aceder a la aplicación. Para ello, se debe de introducir una contraseña 
 ( predeterminada por el fabricante para cada modelo) y si esta es correcta accederemos a la aplicación. En caso de no ser 
 correcta, se dara la opción de volver a introducirla.
